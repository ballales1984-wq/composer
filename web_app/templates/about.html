Music Theory Engine

A comprehensive Python library for music theory analysis designed specifically for guitarists and musicians.

Features

  • Scale Construction (Major, Minor, Modal, Pentatonic, Blues)
  • Chord Building (Triads, 7ths, Extended chords)
  • Arpeggio Generation
  • Progression Analysis with Roman Numerals
  • Integration with music21 and mingus libraries

Technologies

  • Backend: Flask (Python)
  • Core Engine: music_engine
  • Integrations: music21, mingus
  • Frontend: HTML5, CSS3, JavaScript

API Endpoints

  • /api/scales - Scale operations
  • /api/chords - Chord operations
  • /api/progressions - Progression analysis
  • /api/analysis - Advanced analysis
