MUSIC THEORY ENGINE ICON

To add a custom icon to the executable:

1. Create or download a .ico file (icon format)
2. Name it "icon.ico"
3. Place it in the music_engine directory
4. Run the build_exe.py script

The icon should be:
- 256x256 pixels (recommended)
- .ico format
- Square aspect ratio

You can create icons using:
- Online icon generators
- GIMP or Photoshop
- Free icon websites

For a music-themed icon, consider:
- Musical notes (♪ ♫ ♪)
- Guitar silhouette
- Treble clef
- Piano keys

Example free icon websites:
- flaticon.com
- iconfinder.com
- icons8.com

The icon will be embedded in the .exe file and will appear in:
- Desktop shortcuts
- Taskbar when running
- File explorer
- Windows start menu
